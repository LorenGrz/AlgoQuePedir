  <h1>🍔 AlgoQuePedir</h1>

  <p>
    <strong>AlgoQuePedir</strong> es una aplicación tipo marketplace inspirada en PedidosYa,
    que conecta usuarios con comercios para realizar pedidos online de forma rápida, simple y eficiente.
  </p>

  <hr/>

  <h2>🚀 Arquitectura del proyecto</h2>

  <p>La plataforma está compuesta por tres partes principales:</p>

  <h3>👤 App de Usuario</h3>
  <p><strong>Tecnología:</strong> React</p>
  <ul>
    <li>Explorar comercios y productos</li>
    <li>Realizar pedidos</li>
    <li>Hacer seguimiento de órdenes</li>
  </ul>

  <h3>🏪 App de Comercio</h3>
  <p><strong>Tecnología:</strong> Svelte</p>
  <ul>
    <li>Gestión de catálogo de productos</li>
    <li>Administración de pedidos en tiempo real</li>
    <li>Control de operaciones del negocio</li>
  </ul>

  <h3>⚙️ Backend</h3>
  <p><strong>Tecnologías:</strong> Kotlin + Spring Boot</p>
  <ul>
    <li>APIs REST</li>
    <li>Lógica de negocio</li>
    <li>Autenticación de usuarios</li>
    <li>Persistencia de datos</li>
  </ul>

  <hr/>

  <h2>🧠 Características principales</h2>
  <ul>
    <li>Arquitectura desacoplada</li>
    <li>Separación de aplicaciones por tipo de usuario</li>
    <li>Escalable y mantenible</li>
    <li>Comunicación mediante APIs REST</li>
  </ul>

  <hr/>

  <h2>🛠️ Tecnologías utilizadas</h2>
  <ul>
    <li>Frontend Usuario: React</li>
    <li>Frontend Comercio: Svelte</li>
    <li>Backend: Kotlin + Spring Boot</li>
  </ul>

  <hr/>

  <h2>📌 Objetivo</h2>
  <p>
    Brindar una solución moderna y eficiente para la gestión de pedidos online,
    mejorando la experiencia tanto de usuarios como de comercios.
  </p>
